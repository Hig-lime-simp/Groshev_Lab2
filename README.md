# Groshev_Lab2

Простое консольное приложение на C# (.NET 9.0).

## Структура проекта

```
Groshev_Lab2/
├── Groshev_Lab2.sln      # Файл решения
├── Groshev_Lab2/
│   ├── Groshev_Lab2.csproj  # Файл проекта
│   └── Program.cs           # Точка входа программы
```

## Требования

- .NET 9.0 SDK или новее

## Сборка проекта

```bash
cd Groshev_Lab2
dotnet build
```

## Запуск приложения

```bash
cd Groshev_Lab2
dotnet run
```

## Вывод

Приложение выводит:
```
Hello, World!
```

## Разработка

Откройте файл решения `Groshev_Lab2.sln` в Visual Studio или Visual Studio Code с установленным расширением C#.
